PlotWeaver provides production-grade APIs for speech synthesis, dubbing, subtitling, and voice AI agents — purpose-built for African languages and the world's low-resource language communities.

### Why Plotweaver?

Most AI voice systems collapse outside English and a handful of European languages. PlotWeaver is built differently — as long-term infrastructure, not a one-language demo.

### How to Use This Documentation
This reference is designed to get your integration running in minutes. You can navigate through the following core sections:

- Quickstart: Get your API key and process your first media file in under three minutes.

- Authentication: Learn how to securely authenticate your requests using Bearer tokens.

- Dubbing API: Documentation for our AI-driven dubbing engine, featuring voice cloning that preserves the soul of the original performance.

- Subtitle API: Reference for generating accurate, localized subtitles (SRT/VTT) tailored for diverse linguistic regions.

- Language Support: A complete list of supported African languages and their corresponding ISO codes (e.g., yo-NG, sw-KE).
